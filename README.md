# To-Do List Application

A simple and functional to-do list web application built using HTML, CSS, and JavaScript.  
The application allows users to manage daily tasks with priority levels, filters, and persistent storage.



## Features

- Add and delete tasks
- Mark tasks as completed or incomplete
- Assign priority (High, Medium, Low) to tasks
- Filter tasks by status (All, Active, Completed)
- Dark and light mode support
- Due date and time for tasks
- Data persistence using browser localStorage



## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
